# Slide 3: Core Architecture
**How it Works:**
1.  **Ears (Input):** `SpeechRecognition` library listens for commands via Microphone.
2.  **Brain (Processing):** Google's **Gemini 1.5 Flash** LLM understands natural language and context.
3.  **Voice (Output):** `pyttsx3` converts text to speech for a snappy response.
4.  **Hands (Action):** `pyautogui`, `webbrowser`, and `pywhatkit` interact with the OS and Web.
